## APIs
* [Spoonacular](https://spoonacular.com/food-api) - Comprehensive food API for recipes, ingredients, and nutrition info.
* [Open Food Facts](https://world.openfoodfacts.org/data) - A collaborative database of food products from around the world.

## Datasets
* [USDA FoodData Central](https://fdc.nal.usda.gov/) - The definitive source for food nutrient data.
