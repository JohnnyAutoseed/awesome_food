# Contributing to Awesome Food

Contributions are welcome! This list is a community effort, and we rely on you to keep it high-quality and up-to-date.

## Submission Guidelines

To keep the list clean and useful, please follow these rules:

* **Quality over Quantity**: We only want the best resources. If a project is abandoned or an API is broken, please don't submit it.
* **Format**: Use the following format: `* [Resource Name](URL) - A brief, one-sentence description ending with a period.`
* **Alphabetical Order**: Place your link in the correct category in alphabetical order.
* **No Marketing**: Please don't submit your own project unless it is genuinely "awesome" and provides clear value to the community.
* **Check for Duplicates**: Use `Cmd + F` or `Ctrl + F` to make sure the link isn't already there.

## How to Contribute

1. **Fork** the repository.
2. **Clone** your fork to your local machine (or edit directly on GitHub for small changes).
3. **Create a branch** for your update: `git checkout -b add-my-resource`.
4. **Commit** your changes with a clear message: `feat: add [resource name] to [category]`.
5. **Push** to your fork and **submit a Pull Request**.

Thank you for making this resource better for everyone!
